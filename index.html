Bubble Sort

Ordenamiento de burbuja — visualización paso a paso

Python 
lista = [14, 8, 16, 11, 20, 9, 15]
arr   = lista[:]          # copia del original
n     = len(arr)

for i in range(n):
    for j in range(n-i-1):   # ignora los ya ordenados
        if arr[j] < arr[j+1]:  # mayor a la izquierda
            arr[j], arr[j+1] = arr[j+1], arr[j]

arr  # → [20, 16, 15, 14, 11, 9, 8]

// bucle externo i

Controla las pasadas. Cada pasada garantiza que el elemento más pequeño restante "burbujea" hasta su posición final al final del array.

// bucle interno j

Compara pares vecinos arr[j] vs arr[j+1]. Si el de la izquierda es menor, los intercambia. Va de 0 hasta n-i-1 porque los últimos i ya están fijos.

// ¿por qué n-i-1?

Cada pasada deja un elemento fijo al final. Reanalizarlos sería trabajo inútil, así que el límite del bucle interno se reduce en 1 cada pasada.

// orden descendente

El if arr[j] < arr[j+1] mueve el mayor hacia la izquierda. Cambiándolo a > ordenaría de menor a mayor.
